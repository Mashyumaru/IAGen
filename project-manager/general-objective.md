## Key Principles

*   **Client-Side Focus**: Maximize client-side processing and storage to reduce server dependencies and enhance performance.


*   **Accessibility**: Ensure the application is usable by a wide range of users, adhering to accessibility best practices.


*   **Performance**: Optimize for fast loading times and smooth interactions.


*   **Maintainability**: Write clean, readable, and well-organized code following established coding guidelines.


*   **Security (Client-Side)**: Protect sensitive client-side data (like API keys) through appropriate practices.
